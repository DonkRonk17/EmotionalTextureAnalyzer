## 1. Title Card (16:9 Landscape)

**Dimensions:** 1920x1080 pixels
**Filename:** `eta_title_card.png`

**Prompt:**
```
Create a 16:9 landscape title card for "EmotionalTextureAnalyzer", a software tool that analyzes emotional nuance in text.

Visual elements:
- Dark navy background (#0a0e27) with subtle gradient to deeper blue
- Abstract visualization of emotion: flowing gradient waves in colors representing different emotions (warm orange/red for warmth, cool blue for peace, bright yellow for joy, purple for recognition)
- The waves interweave and create an organic texture pattern
- Central focal point: a stylized brain or consciousness symbol made of overlapping color gradients
- Subtle geometric mesh/grid pattern in the background suggesting AI/analysis
- Bright cyan (#00d4ff) accent lines framing the composition

Text: "EmotionalTextureAnalyzer" in clean, modern sans-serif font
Tagline below: "Beyond Sentiment - Into Texture"

Style: Tech-forward, professional, evocative of emotional depth and AI analysis. Premium software aesthetic. No clip art - sophisticated abstract visualization.
```

## 2. Logo Mark (1:1 Square)

**Dimensions:** 512x512 pixels
**Filename:** `eta_logo_mark.png`

**Prompt:**
```
Create a 1:1 square logo mark for "EmotionalTextureAnalyzer".

Visual concept:
- Dark navy background (#0a0e27)
- Central icon: An abstract symbol combining:
  - A stylized heart or circular core representing emotional center
  - Surrounding texture/layers in multiple colors (warm orange, cool blue, soft purple, bright yellow) fading outward like emotional wavelengths or ripples
  - The layers should have a textured, fabric-like or woven quality (representing "texture")
- Bright cyan (#00d4ff) accent glow around the central element
- Clean, minimal, iconic design suitable for app icons

Style: Modern, abstract, professional. Should work at small sizes (32px). No text in this version. The symbol alone should convey emotional depth and analysis.
```

## Usage Notes

1. **Color Palette:**
   - Primary Background: #0a0e27 (dark navy)
   - Accent: #00d4ff (bright cyan)
   - Emotion colors: 
     - Warmth: #ff7f50 (coral orange)
     - Peace: #4169e1 (royal blue)
     - Joy: #ffd700 (gold)
     - Recognition: #9370db (medium purple)
     - Fear: #8b0000 (dark red)
     - Longing: #48d1cc (medium turquoise)

2. **Design Philosophy:**
   - The visual branding should evoke emotional depth without being overwhelming
   - The "texture" concept is key - layers, weaves, gradients
   - Balance between warmth (emotional) and precision (analyzer)
   - Premium, professional, technology-forward

3. **File Formats:**
   - PNG with transparency where applicable
   - High resolution originals, scaled versions as needed
